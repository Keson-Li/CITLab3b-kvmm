Learn Web Coding in 12 Weeks

Web Development Fundamentals is a series of hands-on web code development and related courses leading to a statement of completion from BCIT Computing. Students with no web coding background will learn to write code to develop and manage basic web sites to industry standards. This statement of completion is a subset of the Applied Web Development (AWD) associate certificate in Computing Part-time Studies. This foundation in web development may be completed in as little as one term (12 weeks) part-time. COMP courses are available weeknights, Saturdays and with fast-track day delivery options at our BCIT Downtown Campus near the Granville SkyTrain Station.
